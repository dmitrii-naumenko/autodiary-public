# Видение и цели

## Продуктовая суть

Autodiary — персональный аналитический помощник, который превращает разрозненные жизненные данные в понятные выводы и действия. Система соединяет сигналы из разных доменов (сон, активность, здоровье, эмоции, отношения, продуктивность), находит паттерны и помогает принимать решения на основе фактов, а не ощущений.

## Цели

### Личный помощник

Главная цель — удобный и полезный AI-помощник для повседневных решений:

- **Evidence-first**: выводы опираются на данные, ранние сигналы видны до того, как проблема станет очевидной, гипотезы проверяемы через эксперименты
- **Снижение когнитивной нагрузки**: не «думать что анализировать», а получать 3–7 приоритетов в один клик
- **Кросс-доменный анализ**: соединять данные разных сфер жизни, видеть взаимные влияния
- **Автоматизация**: автоматический сбор метрик, фоновая работа без ручного запуска
- **Долгосрочная память**: база «что работает именно для меня», накопление инсайтов как промежуточный артефакт до встраивания в жизнь
- **Грунтовка знанием**: выводы анализаторов подкрепляются идеями из реально прочитанных книг — личная библиотека как внешнее подтверждение паттерна
- **Открытый доступ к данным**: MCP/API как интеграционный хребет — можно подключать к разным фронтам и моделям

### Демонстрация инженерной зрелости

- Чёткая архитектура, контракт данных, связка LLM + Tools
- Data engineering, product thinking, observability, MLOps-подходы (eval/critic), privacy

### Потенциальный продукт

- **Privacy-first**: личные данные хранятся локально на компьютере пользователя
- **Гибкость**: можно менять набор дневников, источников и метрик; при изменениях — пересчитать исторические данные
- **B2C**: personal analytics assistant — улучшенный дневник + coach
- **B2B ниша**: коучи, психологи, нутрициологи — клиент ведёт дневники, система готовит сводку с доказательствами
- **Уникальность**: evidence-first + privacy-first + multi-domain + контекст-пакеты

## Цепочка ценности

```
Signals → Aggregates → Evidence / Patterns → Insights → Actions
                    ↻ Continual Improvement
```

- **Claude Code Agents**: еженедельные и ежемесячные аналитические отчёты, утренний ритуал `/phrases`, медицинский консультант `/health-consult` по личной истории болезни
- **Grafana**: 10+ дашбордов с privacy-профилями
- **macOS Alerts**: ранние сигналы (HRV deviation, признаки болезни, устаревшие данные)
